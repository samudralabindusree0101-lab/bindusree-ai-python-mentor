# 🤖 AI Python Mentor – Bindusree

## 📌 Overview
AI Python Mentor is an interactive learning assistant that helps beginners understand Python concepts using AI. It provides explanations, examples, quizzes, and feedback.

---

## 🚀 Features
- 🧠 AI-powered explanations (Beginner-friendly)
- 💡 Real-life coding examples
- ❓ Auto-generated quizzes
- ✅ Answer evaluation
- 🎯 Personalized learning experience

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Google Gemini API
- Pyngrok (for sharing app)

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
